Paint Mixer Tanks

Different colors are fed into tank 1, tank 2, and tank 3. The different Colors are then fed into a main tank. The main tank has a mixer, which helps in mixing the different colors. The mixed colors are then drained into a transporting lorry or a storage facility. After the low-level sensor of the main tank is reached, the process starts over again. Timers are used to open the valves in a specified sequence.

This snippet shows the home page. On this page, the operator can start and stop the system, and they can also select the operation mode(either Automatic or Manual Mode). This page has 3 small tanks and one main tank. Each of the 3 tanks has inflow valves and outflow valves. The main tank has a mixer motor and a draining valve. This image shows the alarms page. This page displays all the alarms in case any valve fails.

This snippet shows the trends page. The trend lines represent the process of filling the 4 tanks. When filling the tanks, the trend lines rise, and when draining the tanks, the trend line falls.

This snippet shows the administration page. The page can only be accessed by the administrator. In this page, the administrator can add new users, edit existing users, and delete existing users.
